# 如何构建一个iOS的Framework

## 准备工作

1. 下载模板 `git clone git@github.com:kstenerud/iOS-Universal-Framework.git`

2. 安装模板 （直接运行目录**Fake Framework**下的`install.sh`）

3. 重启Xcode

4. 新建项目时，在**Framework & Library**中可如果看到**Fake Static iOS Framework**就说明已经安装成功了。
